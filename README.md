# ft_linear_regression

## Setup
```bash
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

## Usage
```bash
# Training
$ python src/training.py
# Use -h flag to check for additional options

# Predicting
$ python src/predict.py <mileage>

# Evaluating
$ python src/evaluate.py

```
